Remember

Dissatisfied with the complexity and unnecessary features of many calendar applications, I decided to create a more simplistic program that allows for the viewing and creation of events in a single-page view. The list layout allows you to see all upcoming events at a glance, and the sidebar allows you to create new events by simply tapping on the date. Created in Python with Pythonista for iOS.
